---

## 📦 How to Use This Helm Repository

Add this repo to your Helm client:

```bash
helm repo add thangmai-charts https://ducthangqd1998.github.io/charts
helm repo update
```

Install a chart:

```bash
helm install my-release thangmai-charts/metrics-accumulator
```
